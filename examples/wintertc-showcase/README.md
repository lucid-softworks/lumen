# WinterTC showcase

A single script touring lumen's implementation of the WinterTC **Minimum Common API** (53/53
interfaces): encoding, `structuredClone`, `URL`/`URLPattern`, `crypto`, `Blob`/`File`/`FormData`,
Streams (`TransformStream` + `CompressionStream`), `fetch` against a live server, `WebAssembly`,
and the platform globals.
